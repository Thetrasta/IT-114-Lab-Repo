What is a Class - 
A Class is a template (blueprint) from which objects are made. It defines properties (states) and behaviors (methods)

What is an Object - 
An Object is an instance of a class. You can build several houses from one blueprint. Each house has a different
address, color, size, amount of rooms, etc.

Four Pilars of OOP (object oriented programming):
Encapsulation - Bundling data and methods that operate on that dats within a single unit (class) and restricting access
Inheritance - A mechanism where a new class derives properties and characteristics from an existing class
Polymorphism - The ability of an object to take on many forms; methods can do different things based on the object
